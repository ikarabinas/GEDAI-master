## GEDAI denoising plugin for Matlab
This GEDAI fork includes scripts for usage with custom reference covariance matrices (refCOV). 

Custom refCOV matrices can be generated from individual participant lead fields via
the scripts in the generate_participant_refCOV folder (only for participants with available T1 and/or T2 MRI).
The forward modeling here is done in MNE-python.

See ikarabinas/RELAX for example usage of the RELAX pipeline (forked from NeilwBailey/RELAX) for resting state EEG denoising with GEDAI integration.
