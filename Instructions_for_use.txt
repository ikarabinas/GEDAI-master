This fork of GEDAI includes scripts for usage with custom reference covariance matrices (refCOV). 

Custom refCOV matrices can be generated from individual participant lead fields via
the scripts in the generate_participant_refCOV folder (only for participants with available T1 and/or T2 MRI).
The forward modeing here is done with MNE-python.

See ikarabinas/RELAX for example usage of the RELAX pipeine (forked from NeilwBailey/RELAX) with GEDAI integration.

See original GEDAI repository for info on basic usage in EEGLAB: NeuroTuning Lab [https://github.com/neurotuning]
